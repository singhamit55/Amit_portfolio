# Amit Singh - Portfolio

A modern, responsive portfolio website showcasing my work as an AI/ML Engineer and CSE student.

## ✨ Features

- **Dark/Light Mode Toggle** - Switch between themes with a single click
- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Smooth Animations** - Beautiful reveal animations and transitions
- **CV & Research Papers** - Easy access to download CV and view PDFs
- **Smooth Scrolling** - Native scroll behavior with animated elements
- **Mobile Menu** - Hamburger menu for mobile navigation
- **Modern UI** - Clean, professional design with frosted glass effects

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Advanced styling with animations and transitions
- **JavaScript** - Interactive features and animations
- **Responsive Design** - Mobile-first approach

## 📁 File Structure

```
portfolio/
├── index.html          # Main HTML file
├── style.css           # All styling and animations
├── script.js           # JavaScript functionality
├── CV.pdf              # Resume/CV
└── README.md           # This file
```

## 🚀 Features Breakdown

### Theme Toggle
- Click the moon/sun icon to switch between dark and light modes
- Your preference is saved in browser localStorage

### Navigation
- Fixed navbar with smooth blur effect
- Home, About, Skills, Projects, Research, and Contact sections
- Mobile-responsive hamburger menu

### Modals
- Click-outside to close modals
- Press Escape key to close
- Download buttons for CV and research papers

### Animations
- Scroll-triggered reveal animations
- Hover effects on cards and buttons
- Smooth transitions throughout

## 📱 Responsive Breakpoints

- Desktop: 1024px and above
- Tablet: 768px - 1023px
- Mobile: Below 768px

## 🎨 Color Scheme

### Dark Mode (Default)
- Background: #0a0a0f
- Text: #e8e8f0
- Accent (Gold): #c9a84c

### Light Mode
- Background: #ffffff
- Text: #1a1a24
- Accent: #db770d

## 🔧 How to Use

1. Open `index.html` in your browser
2. Navigate using the menu links
3. Click the theme toggle (top-right) to switch modes
4. Use the hamburger menu on mobile devices
5. Click buttons to view CV or research papers
